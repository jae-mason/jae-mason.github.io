What is the difference between margin, border and padding?

When coding, each visual element basically consists of a box. All these little boxes and the content put into them, along with some styling, make up the web pages we view each day. See the example below

,

Now each of these boxes as seen above are made up of four components.

  • The margin
  • The border
  • The padding
  • The content

Clarification is as follows.

The content space contains the text you read, the links you click and the images/videos you see and hear.
The padding is the space between the content and the border, and the margin is the space outside of the border. The padding sets the distance the content will be from the border box. This allows for spacing and styling.
The border is the containing area limit of which styling can be applied to box i.e. actual borders, background colours. Finally, the margin is the outside area that can be adjusted for formatting spacing of the code element. This area is would not be seen when viewing web pages, it would just appear as blank canvas space.
